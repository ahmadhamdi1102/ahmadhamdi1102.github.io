Proyek

Aplikasi ini merupakan proyek tugas akhir yang saya kerjakan di PT. Indojava Mulia Wisata Cianjur. Di bangun dengan Framework CodeIgniter 3, Bootstrap, JQuery dan MySQL. Terdapat fitur Profil Travel, Pengeloaan Paket, Pemesanan dan Pembayaran. Untuk pengelolaan pembayaran belum terintegrasi dengan payment gateway, hanya sebatas upload struk transfer yang akan dicek kembali oleh admin.

Aplikasi ini dibangun menggunakan Framework Laravel 8, Laravel-Auth, JQuery, Ajax, Bootstrap dan MySQL. Terdapat fitur Pengelolaan Buku dan Pengelolaan Pemesanan Buku.

Aplikasi ini dibangun menggunakan Framework Laravel-Livewire, Bootstrap dan MySQL. Terdapat fitur Pengelolaan Menu dan Transaksi (Cart).

Aplikasi ini dibangun menggunakan React JS, Node Js (Express, Sequelize), MySQL, dan Sweet Alert 2. Terdapat Fitur Auth - JWT, Pengelolaan Produk, dan Pengelolaan Pesanan.
